Open App
A powerful tool to transfer files when you have no way to directly connect two devices: for example you don't possess a public IP and you are in a place where WebRTC is restricted.
About this software

This software allows you to transfer files between two devices, based on Server-Client concept. Node.js has to be installed in the server, and client only needs a modern browser.

This software uses Google Drive™ as a media to put files in and commands to process.

Requirements and Installation
  • A Google account
  • Node.js and npm installed (Server)
  • A modern browser installed (Client)
  1. Download Server
  2. run "unzip 仮.zip"
  3. run "cd 仮"
  4. run "npm i"
  5. Then run "node ." to start server
Privacy Policy

Commands you sent will be stored in the application-specific folder. It won't be shared with anyone.

If you would like to delete those data, visit Google Drive Settings and press Manage Apps on the left pane. Then find FTPoverGoogleDrive to delete its data.
